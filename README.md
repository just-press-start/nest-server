# 100-things

## Description
Open localhost:PORT for swagger.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Create new module with Nestjs
```bash
nest g module core
// creates module

nest g controller coreModule/islands
// creates controller
// creates test file for controller
// adds controller to module file
```
